# QR Checker
An Android application for scanning and extracting data from QR codes from camera and PDF.

## Technologies used
- [Kotlin](https://kotlinlang.org/)
- [Jetpack Compose](https://developer.android.com/jetpack/compose)
- [Material Design 3](https://m3.material.io/)
- [Room](https://developer.android.com/training/data-storage/room)

## Installation
### Option 1: Download APK
You can install a precompiled .apk file from [Releases](https://github.com/LeonovAndreww/QR-Checker/releases)
### Option 2: Build from Source code
Clone this repository ```git clone https://github.com/LeonovAndreww/QR-Checker.git```

Build the project using [Gradle](https://github.com/gradle/gradle)

Open it with [Android Studio](https://developer.android.com)
